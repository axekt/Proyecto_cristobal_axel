## 📌 Descripción
Este es un proyecto de desarrollo web enfocado en la creación de una interfaz de usuario personalizada. Actualmente, el proyecto cuenta con la estructura base de los archivos web (HTML y CSS) y se encuentra en etapa de experimentación y ajuste de los cimientos visuales.

## 🚧 Estado Actual
**Fase de Pruebas (Testing)**
Actualmente estamos probando la estructura inicial del código y los enlaces de estilo. Las funcionalidades principales y el diseño final aún están en proceso de desarrollo.

## 🚀 Próximas Implementaciones (To-Do)
El objetivo principal de las próximas iteraciones es desarrollar una interfaz de perfil híbrida. Para lograr esto, tenemos pendiente implementar:

- [ ] **Inspiración de GitHub:** Adoptar la distribución de repositorios, gráficos de contribución o la forma en que GitHub presenta la actividad del usuario.

- [ ] **Fusión de Interfaces:** Lograr que la interfaz combine la estética de un portafolio técnico (GitHub) con el diseño social y centrado en el usuario de Instagram para la vista final del perfil.
